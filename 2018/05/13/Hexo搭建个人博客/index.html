维护博客

博客的网页在repo leiz2192.github.io中维护, 而博客的md文件可以在repo leiz2192.hexo中维护. 这样数据和网页分离, 便于管理.

而在电脑或系统改变了, git clone leiz2192.hexo后就可以继续维护博客, 而且历史文章不丢失.
